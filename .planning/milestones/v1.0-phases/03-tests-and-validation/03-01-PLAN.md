**4. TestZombiePrevention (TEST-03):**
- Spawn helper subprocess: `exec.Command(os.Args[0], "-test.run=TestHelperProcess", "--", "true")` with `RTX_TEST_HELPER=1`.
- Capture stderr via `cmd.Stderr = &bytes.Buffer{}` to extract PID from `[rtx] spawned PID <n>` log line.
- After `cmd.Run()` returns, read `/proc/<pid>/status`. If file is gone (error from `os.ReadFile`), PASS (child was reaped). If file exists, check `State:` line does NOT contain `Z`.
- Helper function `extractSpawnedPID(t *testing.T, stderr string) int` parses PID from stderr using `strings.Fields` to find the token after `"PID"`.

**5. TestSignalDelivery (TEST-04):**
- Spawn helper subprocess: `exec.Command(os.Args[0], "-test.run=TestHelperProcess", "--", "sleep", "10")` with `RTX_TEST_HELPER=1`.
- Set `cmd.Stdout = os.Stdout` and `cmd.Stderr = os.Stderr` (signal test does not need to capture output).
- `cmd.Start()`, then `time.Sleep(200 * time.Millisecond)` to allow `process.Run()` to register `signal.Notify`.

- `cmd.Wait()` and assert `errors.As(err, &exitErr)` with `exitErr.ExitCode() == 143` (128 + SIGTERM(15)).
