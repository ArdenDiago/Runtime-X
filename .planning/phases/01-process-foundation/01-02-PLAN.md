Create the directory cmd/rtx/ and the file cmd/rtx/main.go with package main. Implement exactly:

```go
package main

import (
	"flag"
	"fmt"
	"os"

	"runtimex/internal/process"
)

func main() {
	// EXIT-02: os.Exit only in main() so deferred cleanup in run() executes.
	os.Exit(run())
}

func run() int {
	// Global flags (before subcommand)
	verbose := flag.Bool("v", false, "verbose output")
	flag.BoolVar(verbose, "verbose", false, "verbose output")
	flag.Usage = func() {
		fmt.Fprintf(os.Stderr, "usage: rtx [flags] run <command> [args...]\n\n")
		fmt.Fprintf(os.Stderr, "flags:\n")
		flag.PrintDefaults()
	}
	flag.Parse()

	args := flag.Args()
	if len(args) == 0 {
		flag.Usage()
		return 1
	}

	switch args[0] {
	case "run":
		// CLI-01: rtx run <command> [args...]
		if len(args) < 2 {
			fmt.Fprintf(os.Stderr, "[rtx] error: 'run' requires a command\n")
			fmt.Fprintf(os.Stderr, "usage: rtx run <command> [args...]\n")
			return 1
		}
		// CLI-02: PID is logged inside process.Run() immediately after spawn
		return process.Run(args[1], args[2:])
	default:
		fmt.Fprintf(os.Stderr, "[rtx] unknown subcommand: %s\n", args[0])
		fmt.Fprintf(os.Stderr, "usage: rtx [flags] run <command> [args...]\n")
		return 1
	}
}
```

CRITICAL constraints from research:
- `main()` MUST contain ONLY `os.Exit(run())` — no logic in main() (EXIT-02 / Pitfall 5).
- `run()` returns int — never calls os.Exit() directly inside run() (deferred cleanup would be skipped).
- `flag.Parse()` handles global flags; `flag.Args()` gives everything after flags (including the subcommand).
- Import path is `runtimex/internal/process` (module name from go.mod is `runtimex`).
- The `verbose` flag is defined for future use but not yet wired to anything — that's intentional for Phase 1.
- Do NOT use cobra, urfave/cli, or any other external CLI library — stdlib `flag` only per research constraints.
