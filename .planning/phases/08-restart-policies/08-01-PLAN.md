- Edit `internal/scheduler/types.go`.

### 3. Update `validTransitions` (RST-04)
- Add transitions from `StateRunning` → `StateRestarting`.
- Add transitions from `StateRestarting` → `StateStarting` (successful backoff).
- Add transitions from `StateRestarting` → `StateStopping` (interrupted by `Stop()`).
- Add transitions from `StateRestarting` → `StateFailed` (max retries).

### 4. Update `ManagedProcess` (RST-04)
- Add `restartCancelCh chan struct{}` to `ManagedProcess`.
- This channel will be created on start and closed on stop to signal any pending backoff to exit.
