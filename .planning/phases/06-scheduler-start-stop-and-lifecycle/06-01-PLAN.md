Replace stubs in `lifecycle.go` with full implementations. Follow the locked patterns from CONTEXT.md and RESEARCH.md exactly:

**Start(name string) error:**
1. Acquire s.mu.Lock()
2. Look up process by name — return ErrNotFound if missing
3. Check state: allow Idle, Stopped, Failed. Return ErrAlreadyRunning for Running. Return descriptive error for Starting/Stopping.
4. Transition to StateStarting (using `transition(mp, StateStarting)`)
5. Reset runtime metadata: mp.StartedAt = time.Now(), mp.StoppedAt = time.Time{}, mp.ExitCode = 0
6. Capture `def := mp.Def` as local copy
7. **s.mu.Unlock()** — CRITICAL: release write lock before cmd.Start() per STATE.md arch decision
8. Build fresh `*exec.Cmd` — `exec.Command(def.Command, def.Args...)`, set Dir, Env, SysProcAttr with Setpgid:true
9. Attach StdoutPipe and StderrPipe — on pipe error, relock, transition to Failed, unlock, return error
10. Call cmd.Start() — on error, relock, transition to Failed, unlock, return wrapped error
11. Re-acquire s.mu.Lock(), transition to StateRunning, set mp.cmd = cmd, s.mu.Unlock()
12. Launch `go captureOutput(mp.logs, stdoutPipe, StreamStdout)`
13. Launch `go captureOutput(mp.logs, stderrPipe, StreamStderr)`
14. Launch `go monitorProcess(s, mp, cmd)`
15. Return nil

**captureOutput(lb *logBuffer, r io.ReadCloser, stream Stream):**
- Create bufio.Scanner with scanner.Buffer(make([]byte, 4096), maxLineBytes) where maxLineBytes = 8*1024
- Loop scanner.Scan(), calling lb.Write(LogEntry{Timestamp: time.Now(), Stream: stream, Text: scanner.Text()})
- No error handling on scanner.Err() — EOF and pipe close are expected and benign

**monitorProcess(s *Scheduler, mp *ManagedProcess, cmd *exec.Cmd):**
- Call `err := cmd.Wait()` — blocks until process exits AND pipe goroutines finish
- Acquire s.mu.Lock(), defer s.mu.Unlock()
- Set mp.StoppedAt = time.Now()
- Read exit code from cmd.ProcessState.ExitCode() (guard: if cmd.ProcessState != nil, else -1)
- Set mp.ExitCode = code
- If mp.State == StateStopping: transition to StateStopped (stop was requested)
- Else if err == nil or code == 0: transition to StateStopped (clean exit)
- Else: transition to StateFailed (crash)
- If mp.doneCh != nil: close(mp.doneCh), set mp.doneCh = nil
