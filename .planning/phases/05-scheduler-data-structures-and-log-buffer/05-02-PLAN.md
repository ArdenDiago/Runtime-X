  <behavior>
    **types.go — All type definitions and constants:**

    RestartMode type (string):
    - RestartAlways = "always", RestartOnFailure = "on-failure", RestartNever = "never"

    RestartPolicy struct:
    - Mode (RestartMode), MaxRetries (int), Delay (time.Duration)

    ProcessDef struct (locked shape from CONTEXT.md):
    - Name (string), Command (string), Args ([]string), Env ([]string — KEY=VALUE pairs), WorkDir (string — "" = inherit)
    - RestartPolicy (RestartPolicy), DependsOn ([]string — ignored until Phase 7), LogBufferSize (int — 0 = default 1000)

    State type (int) with iota constants:
    - StateIdle (0), StateStarting (1), StateRunning (2), StateStopping (3), StateStopped (4), StateFailed (5)
    - String() method returning lowercase names: "idle", "starting", "running", "stopping", "stopped", "failed"

    State transition table (package-level var validTransitions map[State][]State):
    - StateIdle -> {StateStarting}
    - StateStarting -> {StateRunning, StateFailed}
    - StateRunning -> {StateStopping, StateFailed}
    - StateStopping -> {StateStopped, StateFailed}
    - StateStopped -> {StateStarting} (allows restart)
    - StateFailed -> {StateStarting} (allows retry)

    ErrInvalidTransition sentinel error.
    canTransition(from, to State) bool — unexported helper.
    transition(mp *ManagedProcess, to State) error — unexported method that validates via canTransition and sets mp.State, or returns fmt.Errorf wrapping ErrInvalidTransition with from/to context.

    ManagedProcess struct:
    - Def (ProcessDef), State (State), StartedAt (time.Time), StoppedAt (time.Time), ExitCode (int), RestartCount (int), logs (*logBuffer — unexported)

    **scheduler.go — Scheduler struct and methods:**

    Package-level: validName = regexp.MustCompile(`^[a-z0-9][a-z0-9-]*$`)
    Sentinel errors: ErrNotFound, ErrAlreadyExists, ErrNotStopped

    Scheduler struct:
    - mu (sync.RWMutex), processes (map[string]*ManagedProcess)

    New() *Scheduler — constructor, initializes map.

    Register(def ProcessDef) error:
    - Validate name with validateName() — return error if invalid
    - Default LogBufferSize to 1000 if <= 0
    - Acquire write lock
    - Check for duplicate name -> return fmt.Errorf("%w: %s", ErrAlreadyExists, def.Name)
    - Create ManagedProcess{Def: def, State: StateIdle, logs: newLogBuffer(def.LogBufferSize)}
    - Store in map

    Remove(name string) error:
    - Acquire write lock
    - Check exists -> ErrNotFound
    - Check state is Stopped, Idle, or Failed -> ErrNotStopped
    - Delete from map

    Get(name string) (*ManagedProcess, error):
    - Acquire read lock
    - Lookup -> ErrNotFound or return pointer

    List() []*ManagedProcess:
    - Acquire read lock
    - Return new slice of all values

    Logs(name string) ([]LogEntry, error):
    - Acquire read lock to find ManagedProcess
    - Release read lock BEFORE calling mp.logs.Lines() (log buffer has its own mutex — do not hold scheduler lock while accessing it)
    - Return the snapshot

    validateName(name string) error — unexported, uses validName regexp.

    Test cases (input -> expected output):
    - Register valid def -> no error, Get returns it with StateIdle
    - Register duplicate name -> ErrAlreadyExists
    - Register empty name -> error
    - Register invalid name "-foo" -> error
    - Register invalid name "FOO" -> error
    - Register invalid name "under_score" -> error
    - Register valid names: "my-app", "web1", "a" -> all succeed
    - Register with LogBufferSize 0 -> defaults to 1000, verify via Logs returning nil (empty buffer)
    - Remove idle process -> success, Get returns ErrNotFound
    - Remove nonexistent -> ErrNotFound
    - Remove running process (manually set state) -> ErrNotStopped
    - List empty scheduler -> empty slice
    - List after 3 registrations -> 3 entries
    - Logs on registered process (no writes) -> nil
    - Logs on nonexistent -> ErrNotFound
    - State transitions: Idle->Starting (ok), Idle->Running (error), Starting->Running (ok), Starting->Failed (ok), Running->Stopping (ok), Running->Failed (ok), Stopping->Stopped (ok), Stopped->Starting (ok), Failed->Starting (ok)
    - State String(): each state returns correct lowercase string
    - Concurrent: multiple goroutines calling Register/Get/List simultaneously -> go test -race passes
